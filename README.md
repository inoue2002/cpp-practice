## 動作確認方法

### Mac / Linux 環境での実行方法

1. コンパイル方法
   ```
   g++ 1.cpp -o 1
   ```
   このコマンドでC++ソースファイル（1.cpp）をコンパイルし、実行ファイル（1）を生成します。

2. 実行方法
   ```
   ./1
   ```
   生成された実行ファイルを上記コマンドで実行します。

### Windows環境での実行方法

1. コンパイル方法
   ```
   g++ 8.cpp -o 8 -fexec-charset=CP932
   ```
   このコマンドでC++ソースファイル（8.cpp）をコンパイルし、実行ファイル（8.exe）を生成します。

2. 実行方法
   ```
   ./8.exe
   ```
   生成された実行ファイルを上記コマンドで実行します。
